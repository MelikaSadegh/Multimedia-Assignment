About This Project

Project Overview

This is an advanced deep learning system for multi-class skin cancer classification, implementing the research paper "Multi-Class Skin Cancer Classification Using Vision Transformer Networks and Convolutional Neural Network-Based Pre-Trained Models".

The system uses state-of-the-art Vision Transformer architecture along with multiple CNN models to achieve high accuracy in detecting 7 different types of skin lesions.
